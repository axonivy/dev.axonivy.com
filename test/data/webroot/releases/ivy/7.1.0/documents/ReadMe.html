Upgrading from an older version

Xpert.ivy becomes Axon.ivy

With version 5.1 the product was renamed from Xpert.ivy to Axon.ivy.

Therefore, don't be confused if the product name of all the versions mentioned below (3.x / 4.x / 5.x) is Xpert.ivy and not Axon.ivy.

Upgrading from 3.x

Projects which are compatible with Xpert.ivy 3.9 or higher can be imported to Axon.ivy Designer 7.1 using the import wizard. Read the chapter Help > Designer Guide > Projects > Importing a Project to learn more about this new feature.

Upgrading from 4.x/5.x/6.x

Read the Migration Notes to learn about the necessary steps to convert a project from an older 4.x, 5.x or 6.x Designer to 7.1.
