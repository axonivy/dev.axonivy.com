Upgrading from an older version

Xpert.ivy Server becomes Axon.ivy Engine

With version 5.1 the product was renamed from Xpert.ivy Server to Axon.ivy Engine.

Therefore, don't be confused if the product name of all the versions mentioned below (3.x / 4.x / 5.x) is Xpert.ivy Server and not Axon.ivy Engine.

Upgrading from 3.x

Xpert.ivy Server 3.x cannot be upgraded to 6.0. However, 3.9 projects can be converted into 7.1 projects using the Axon.ivy Designer 7.1. Read the chapter Help > Designer Guide > Projects > Importing a Project of the Designer to learn more about this new feature.

Upgrading from 4.x/5.x/6.x

Read the Migration Notes to learn about the necessary steps to migrate from an older 4.x, 5.x or 6.x to the latest version 7.1.
