Axon.ivy IvyAddOns

The IvyAddOns project is provided as a convenience utility project for Axon.ivy developers. It contains various reusable components, such as:

  • Dynamic Dialog turn your business object directly into a GUI
  • DocFactory generating automatically documents like serial letters with the help of documents models like Microsoft Office Templates
  • Common Rich Dialogs general and customizable implementation of often-used user-interaction dialogs
  • File Management System allows sharing, editing, versioning, copying and printing of files in the Ivy RIA Applications
  • And many more Java Helper Classes, Libraries, Wait for an asynchronous process
  • Download 4.3
  • Download 5.0
Newer versions are directly included in the Axon.ivy Designer & Engine

Axon.ivy Portal

The Axon.ivy Portal is intended as a starting point on your ivy process server. It helps to develop and publish your project(s) together with extended workflow and administration functionality. Versions from 6.0 are directly included in the Axon.ivy Engine.

  • Download 5.0

Axon.ivy Monitor

This is an additional module of the Axon.ivy Digital Business Platform. It is designed for monitoring your process applications in real-time. Axon.ivy Monitor stands for Business Activity Monitoring (BAM) and features different views on your processes:

  • Dashboard Summary
  • Process Overview
  • Role Overview
  • Each User
Contact us for more information.
