## Engine Cockpit {#engineCockpit}

The engine cockpit has been improved with many small features that makes the lives of operators easier. 

- **Migration Wizard**: The completely overhauled migration wizard supports migrating an Engine from one version to another, be it a LTS to next LTS migration or a migration from one update release to another.
- **Health Check**: The new Engine health check service analyzes with a lot of different checks the health of your Engine and warns if your Engine is in an unhealthy situation.
- **SSL / TLS Tester**: Manage your keys and certificates stored in the key and trust store in the new SSL view. Use the TLS tester to test connections to databases, REST, and SOAP services, and directly import missing certificates.
- **Monitoring**: Analyze what is happening inside your Engine by using eight new monitor views: System Database Info, Class Histogram, Flight Recorder, Notifications, Jobs, Health, Documents, Start Events, and Intermediate Events. 
- **Security Export**: Export all users, roles, the roles of users, and permissions to Excel files for audits or documentation.
- **Properties and Features**: Edit the properties or features of REST clients or SOAP web services.
